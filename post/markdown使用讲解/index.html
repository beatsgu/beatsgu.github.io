1.标题

Markdown支持6级标题，使用#符号表示：

# 一级标题（#）
## 二级标题（##）
### 三级标题（###）
#### 四级标题（####）
##### 五级标题（#####）
###### 六级标题（######）

2.引用

使用>表示引用：

这是一段引用

这是嵌套引用

> 这是一段引用
> 
> > 这是嵌套引用

3.列表

有序列表

使用数字加点号：

  1. 打开冰箱
  2. 把大象放进去
  3. 关上冰箱
1. 打开冰箱
2. 把大象放进去
3. 关上冰箱

无序列表

可以使用*、-或+：

  • 使用星号（*）

  • 第二项

  • 使用短横线（-）

  • 第二项

  • 使用加号（+）

  • 第二项

任务列表

  • 已完成的任务
  • 未完成的任务
  • 另一个未完成的任务
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个未完成的任务

4.代码

行内代码

使用反引号包围：console.log("Hello World")

`console.log("Hello World")`

代码块

使用三个反引号，可指定语言：

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

15.快捷键提示

  • 粗体：Ctrl/Cmd + B
  • 斜体：Ctrl/Cmd + I
  • 代码：Ctrl/Cmd + `
  • 链接：Ctrl/Cmd + K
  • 图片：Ctrl/Cmd + Shift + I

16.最佳实践

  1. 保持一致性：在整个文档中使用相同的格式约定
  2. 适度使用格式：不要过度格式化，保持内容可读性
  3. 预览检查：写完后预览确保格式正确
  4. 文件命名：使用.md或.markdown扩展名

17.常用工具推荐

  • 编辑器：Typora、Mark Text、VS Code
  • 在线编辑器：Dillinger、StackEdit
  • 转换工具：Pandoc
  • 预览工具：Marked、各种浏览器插件
